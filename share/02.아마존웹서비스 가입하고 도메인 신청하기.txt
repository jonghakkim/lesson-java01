- aws(아마존 웹 서비스) 가입하기
  
  => http://aws.amazon.com/kr
  
  => 상단 배너 "무료 체험하기" 또는 우상단 "콘솔에 로그인" 버튼 클릭
  
  => 새로 가입하기 위해 “새 사용자입니다”에 체크하고 이메일 또는 전화번호 입력
  
  => 추가 정보 입력한 뒤 "계정 생성"버튼 클릭
  
  => 상단 "개인계정"에 체크한 뒤 개인정보 입력
  
  => 카드정보 입력 후, 전화인증 절차 반드시 거칠 것
  
  => 계획 지원은 "기본" 선택
    
  ** 아마존에 처음 가입하면 t2.micro타입의 EC2 인스턴스, 1년간 무료
     - “1년 동안”이라는 것은 월 750시간 무료를 의미한다.
     - 인스턴스 1대 24시간씩 31일이면 744시간
     - 주의사항!!!!!
       => 인스턴스를 2대 이상 돌리기 시작하면 시간이 배로 차감됨.
       => 아마존 웹서비스는 서버를 인스턴스라는 명칭으로 부름


- 도메인 신청하기
  => 가비아 접속 (https://www.gabia.com/)
  => 회원가입하거나 페이스북 계정으로 로그인
  => 상단 "도메인 - 도메인 검색" 메뉴 클릭
     => 도메인 검색해서 선택, 신청하기 
     => 요금 결제 진행
        - 1000원 미만이기 때문에 휴대폰 소액결제만 가능함

     ** 구글 알파벳이라는 회사 도메인 주소 : abc.xyz 
        => 구글에서 .xyz 도메인을 처음 사용한 이후, 개발자들이 많이 사용하는 도메인이다.
        
     ** 추가 사항 
        => 필수사항 아님. 안해도 됨
        => 로그인 상태에서 회원정보 수정으로 이동
        => "임시비밀번호 발급" 클릭
        => 이메일 확인 한 후, 비밀번호 변경
        => 본인 정보 수정
