====> sftp setting
http://sys145.tistory.com/4
  - "Tools - Deployment - Browse Remote Host"
  - "Remote Host" 화면에서 [...] 클릭
  - Connection 탭 설정
    => SFTP host : ip주소, 호스트명
    => Port : 포트주소 (Putty : 22)
    => Root path : 서버 path
    => User name : 계정명
    => Auth type : key로 인증
    => Private key file : key 위치 (abc.pem)
  - Mapping 탭 설정
    => 동기화 디렉토리 설정
  - "Excluded Paths" 탭 설정
    => 비동기화 디렉토리 설정
